# Moto Saldo App

Gerenciador de ganhos, gastos, tempo e distancia para motoboys autônomos.

## Funcionalidades

- [ x ] Registro para novos usuários
- [ x ] Autenticação e login de usuarios
- [ x ] configuração de usuário: (aplicativos que trabalha, quanto tempo pretende
- [ x ] trabalhar mensalmente e quanto pretende fatural mensalmente).
- [ x ] Cálculo de faturamento, distancia percorrida e tempo de corrida por aplicativo.
- [ ] Registrar gastos (com gasolina ou diversos) [em desenvolvimento]
